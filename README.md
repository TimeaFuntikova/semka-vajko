# semka-vajko
Semestrálna práca na predmet VAII

To see the Svelte app:

1. In terminal - change to the project directory:
 ```bash
cd svelte-app
```
2. Install the project dependencies using npm:
 ```bash
npm install
```
3. In the project directory, run the following command to start the development server:
 ```bash
npm run dev
```
